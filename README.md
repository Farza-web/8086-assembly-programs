# 8086 Assembly Programs

## 📘 Description

This repository contains a collection of basic 8086 assembly language programs developed for microprocessor laboratory coursework. The programs focus on fundamental concepts such as array manipulation, loop control, arithmetic operations, and problem-solving using low-level programming.

---

## 🎯 Objectives

* Understand 8086 assembly language basics
* Practice working with arrays and memory addressing
* Learn loop control using registers (CX)
* Perform arithmetic and logical operations
* Build problem-solving skills in assembly language

---

## 📂 Programs Included

1. Sum of elements in an array
2. Find the largest number in an array
3. Count even numbers in an array
4. Reverse an array
5. Display numbers from 1 to 10
6. Sum of first 10 natural numbers
7. Factorial calculation
8. Search an element in an array
9. Multiplication table of 5

---

## ⚙️ Technologies Used

* 8086 Assembly Language
* DOS-based environment (org 100h programs)

---

## ▶️ How to Run

1. Install an 8086 emulator (e.g., EMU8086)
2. Open any `.asm` file
3. Compile and run the program
